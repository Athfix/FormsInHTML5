  • Forms in HTML5

    Well forms are essential things to collect user information such as name, email, phone, password to give them customized app for their usage and these forms are used in login, sign in pages as you have watched in many of them. Well these forms are built in with inputs and many tags which helps to create user required input type. Well we have autocomplete attribute if it is on then the form will fill up everything automatically which are filled in previous form else not. Different things included in forms:

    1. Input
    2. select and option
    3. label
    4. text area
    5. Button
    6. Fieldset
    7. Legend
